# kawayiLive
多端适配的直播平台（直播，聊天室，弹幕，礼物系统）  

***
## 初步思路
  1. 视频模块：直播编解码，弹幕加载  
  2. 聊天室模块：多人了聊天室  
  3. 礼物模块：充值与消费  
  4. 用户中心：  
  4. 总控制服务：负责与前端的所有交互，所有中心为总控制提供能力，总控制负责组织业务，构建VO

## 技术架构图

```sequence
对象A->对象B: 对象B你好吗?（请求）
Note right of 对象B: 对象B的描述
Note left of 对象A: 对象A的描述(提示)
对象B-->对象A: 我很好(响应)
对象A->对象B: 你真的好吗？
```
